## 项目介绍

本项目是[闪鲤电商小程序](https://www.sansanll.com)私有化部署方案. 组成部分:

* 服务端: 提供二进制部署文件
* 管理后台端: 提供一个公共web后台及压缩后的静态文件
* 小程序端: 提供小程序源代码
